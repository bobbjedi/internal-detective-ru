# Внутренний детектив

Настольный/разговорный «тренажёр мышления» для детей (ориентир 8–10): ребёнок замечает внутренние правила (интерналы), проверяет их и придумывает более умную/безопасную версию.

## Что в репозитории
- **`RAW.md`**: сырой поток/черновик (не трогаем).
- **`MANUAL.md`**: причёсанное описание концепта и правил.
- **`cards/`**: печать карточек.
  - **`cards/cards.json`**: единый JSON со всеми карточками и персональными вопросами.
  - **`cards/print.html`**: печать всей колоды (по 6 карточек на A4).
  - **`cards/cards.css`**: стили печати.
  - **`cards/cards.js`**: рендер (читает `cards.json`).
- **`Gemini_Generated_Image_ckqx18ckqx18ckqx.png`**: картинка-водяной знак на карточках.
- **`sheet-6-cards-a4.html`**: отдельный пример листа на 6 карточек (самодостаточный HTML).
- **`parents.html`**: памятка для родителей (удобно распечатать).

## Как распечатать карточки
Файл для печати: `cards/print.html`.

Важно: `print.html` загружает `cards.json`, поэтому открывай через простой локальный сервер (а не `file://`).

Пример (Python):

```bash
cd "Внутренний детектив"
python3 -m http.server 8000
```

Дальше открой в браузере `http://localhost:8000/cards/print.html` и печатай:
- масштаб **100%**
- **без** headers/footers
- **включить** background graphics (чтобы фон печатался)
- печать **односторонняя**

## Где править тексты
- **Карточки и вопросы**: `cards/cards.json`
- **Прозрачность фона**: `cards/cards.json → meta.bgOpacity`
